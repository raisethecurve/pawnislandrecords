Project Identity

World and Sound

Rhea Mauro is Pawn Island Records' soul-rooted singer-songwriter project. The project began in a soul lane and gradually moved toward intimate singer-songwriter material, where voice, feeling, memory, and plainspoken emotional detail became the center.

Rhea Mauro should not be primarily framed as a country project. The country single was an experiment, and any country language should be used narrowly when discussing that specific release.

The broader project belongs closer to soul-influenced singer-songwriter music: warm, human, reflective, and emotionally direct. Aura Farming can be described through home, identity, tenderness, and domestic memory without making modern country the core identity.

Story Angles

  • A soul-rooted project that evolved into singer-songwriter material.
  • Warm, intimate songs built around memory, home, tenderness, and emotional clarity.
  • Country appears as a release-specific experiment, not the whole project identity.
